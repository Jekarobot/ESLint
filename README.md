# ESLint

## 📋 Описание

ESLint — инструмент для статического анализа кода JavaScript. Помогает находить и исправлять проблемы в коде, обеспечивая соблюдение стандартов кодирования и предотвращая потенциальные ошибки.

## 🛠️ Технологический стек

- **JavaScript** — основной язык проекта
- **Node.js** — среда выполнения
- **ESLint** — инструмент линтинга

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/username/eslint.git
   cd eslint
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите ESLint:
   ```bash
   npx eslint src/app.js
   ```

## 💻 Примеры использования

**Проверка одного файла:**
```bash
npx eslint src/app.js
```

**Проверка всех файлов в директории:**
```bash
npx eslint src/
```

**Автоматическое исправление ошибок:**
```bash
npx eslint src/app.js --fix
```
